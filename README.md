# ü§ù Caring for Carers ‚Äì Autism Caregiver App

A **mobile application** designed to support caregivers of children with Autism Spectrum Disorder (ASD). The app helps reduce emotional, mental, and physical stress by offering routine scheduling, behaviour logging, mood tracking, and self-care reminders. It also includes an **AI-powered companion** for emotional support.  

Developed by **Group 63** as part of **PUSL2021 ‚Äì Computing Group Project** using **Flutter**, **Firebase**, and **TensorFlow Lite**.

---

## üéØ Key Features

- **Caregiver Dashboard**: Track daily tasks, moods, and child behaviour patterns  
- **Child Dashboard**: Visual task progress and activity tracking  
- **Behaviour & Mood Logging**: Flexible forms and charts to monitor changes  
- **AI Companion**: TensorFlow Lite-based emotional support for caregivers  
- **Task Management**: Routine creation with text & image visualization  
- **Secure Authentication**: Firebase Authentication for caregivers and children  
- **Real-time Data Sync**: Firebase Firestore updates across devices  
- **Self-care Reminders**: Notifications and diary tools for caregivers
